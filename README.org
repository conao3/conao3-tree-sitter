* Overview

A collection of custom [[https://tree-sitter.github.io/tree-sitter/][Tree-sitter]] grammars and corresponding Emacs major modes. Tree-sitter provides fast, accurate parsing for programming languages, enabling better syntax highlighting, code navigation, and structural editing.

* Grammars and Modes

| Language   | Tree-sitter Grammar                  | Emacs Mode                       |
|------------+--------------------------------------+----------------------------------|
| JSON       | [[https://github.com/conao3/tree-sitter-conao3-json][tree-sitter-conao3-json]]              | [[https://github.com/conao3/elisp-conao3-json-mode][conao3-json-ts-mode]]              |
| JavaScript | [[https://github.com/conao3/tree-sitter-conao3-javascript][tree-sitter-conao3-javascript]]        | -                                |
| Clojure    | [[https://github.com/conao3/tree-sitter-conao3-clojure][tree-sitter-conao3-clojure]]           | -                                |
| Bash       | [[https://github.com/conao3/tree-sitter-conao3-bash][tree-sitter-conao3-bash]]              | -                                |
| SQL        | [[https://github.com/conao3/tree-sitter-conao3-sql][tree-sitter-conao3-sql]]               | -                                |
